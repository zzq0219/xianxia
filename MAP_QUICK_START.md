# 🗺️ 仙侠世界地图 - 快速上手指南

## 🚀 快速开始

### 1. 查看演示
直接打开 `world-map-demo.html` 文件即可看到完整的地图演示效果。

### 2. 在项目中使用

地图已经自动集成到你的游戏中！点击游戏底部的 **地图** 按钮即可打开。

## 🎮 操作指南

### 基础操作
- **🖱️ 滚轮**：放大/缩小地图（0.5x - 3x）
- **✋ 拖拽**：按住鼠标左键拖动地图
- **👆 点击地点**：查看地点详情并前往

### 地图元素

#### 地点图标
- 🏛️ **宫殿形状** = 宗门（Sect）
- 🏘️ **建筑群** = 城市（City）  
- ⛰️ **山峰轮廓** = 山脉（Mountain）
- 🌳 **树木形状** = 森林（Forest）
- 🌊 **水流曲线** = 河流（River）
- 🕳️ **洞穴形状** = 洞府（Cave）

#### 视觉提示
- ⭐ **金色星标** = 重要地点
- 💫 **脉冲光晕** = 当前位置
- ✨ **放大效果** = 鼠标悬停

## 💡 使用技巧

### 快速导航
1. 点击地图上的任意地点
2. 右侧面板会显示详细信息
3. 点击"前往此地"按钮开始旅程

### 寻找重要地点
- 带有 **⭐ 金色星标** 的都是重要地点
- 这些地点通常有特殊事件或任务

### 探索新区域
- 每个区域都有独特的颜色光晕
- 建议按照：东荒 → 中州 → 其他区域的顺序探索

## 🔧 自定义选项

### 在代码中调整

#### 修改缩放范围
打开 [`WorldMapSVG.tsx`](components/WorldMapSVG.tsx:24)，找到：
```tsx
setScale(prev => Math.max(0.5, Math.min(3, prev * delta)));
```
修改 `0.5`（最小缩放）和 `3`（最大缩放）

#### 添加新地点
在 [`locations.ts`](locations.ts:15) 中添加：
```tsx
{
  id: 'my_location',
  name: '我的地点',
  type: 'Sect',
  region: '东荒',
  x: 300,
  y: 400,
  description: '这是我的自定义地点',
  isMajor: true
}
```

#### 修改区域颜色
在 [`WorldMapSVG.tsx`](components/WorldMapSVG.tsx:47) 的 `getRegionGradient` 函数中修改颜色值。

## 🎯 常见问题

### Q: 如何重置地图视图？
A: 刷新页面或重新打开地图即可重置到初始状态。

### Q: 地图卡顿怎么办？
A: 
1. 减少浏览器标签页数量
2. 关闭其他占用GPU的程序
3. 尝试缩小地图（减少可见元素）

### Q: 如何截图保存地图？
A: 
1. Windows: `Win + Shift + S`
2. Mac: `Cmd + Shift + 4`
3. 或者使用浏览器的开发者工具导出SVG

### Q: 能否添加更多地点？
A: 当然可以！参考上面的"添加新地点"部分。
